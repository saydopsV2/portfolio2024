Information de l'interface Homme/Machine

  • Langage: C# / python / sql
  • Date du projet: 05 juin 2023

Création d'une application avec un interface Homme/Machine.

  • Conception d’une application en C# permetant de visualiser les lieux de pollution où est implantée l’entreprise.
  • Création d’une base de données SQL Server.
  • Traitement et visualisation des données via python grâce aux librairies pandas et SQLAlchemy.ation est faite pour accompagner l’utilisateur au maximum afin qu’il ne soit pas perdu et réalise des actions incohérentes ou impossibles.
